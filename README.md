# E-Commerce Order Management System

A RESTful API built with **Spring Boot** and **MySQL** to manage e-commerce orders. This system allows users to create orders, manage order items, and handle customer and shipping details dynamically.

## Features
* **Create Orders:** Generate new orders with multiple items (e.g., T-Shirts, Jeans) for specific customers.
* **Retrieve Data:** Fetch detailed order information, including customer and shipping details.
* **Update Orders:** Modify order details or update specific item quantities (e.g., changing quantity from 1 to 2).
* **Manage Items:** Add new items to existing orders or remove specific items.
* **Delete Operations:** Cancel entire orders or remove individual line items.

## Tech Stack
* **Language:** Java (JDK 21)
* **Framework:** Spring Boot (Web, JPA, Data)
* **Database:** MySQL
* **Build Tool:** Maven
* **Testing:** Postman
